Do It Right

What it is..

Do It Right is a fitness app that leverages augmented reality. It creates a three dimensional model of a person portraying how a particular exercise should be done.

How it would help..

By observing the 3D augmented model the user can check if he is performing the exercise correctly or not

Doing exercises correctly greately reduces the chances of injuries
